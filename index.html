Mac环境下Github Pages + Hexo搭建个人博客

前言

入职公司一年了，工作算是比较稳定。一直以来都想要建立一个自己的博客，但是一直都停留在想的阶段！直到前一段时间在微信上看我为什么写博客这篇文章的时候，才算是真的坚定了这份心！也正是由于博主的推荐，我选择了使用Github Pages和Hexo来搭建个人博客，在这里，主要是做个记录，方便以后自己查看或者他人借鉴！

基础准备

在搭建博客的过程当中，需要以下工具和网站的支持：
Homebrew
Git
Github
nvm
node.js
hexo

  • Homebrew作为Mac上面套件管理神器，早就在配置ruby环境的时候安装上了。如果没有，一句简单的命令足矣
  • Git作为世界上最先进的分布式版本控制系统，已经在Xcode安装的时候内置了～如果没有，同样的一句简单的命令
  • 作为一个开发人员，加上网站是托管在Github上面，一个Github帐号是必须的了。登录github.com，创建一个仓库“github用户名.github.io”。由于我们使用Github Pages搭建博客，所以必须遵循这样的命名规范。
  • nvm作为node.js的版本管理器，安装方式也很简单，通过一句命令即可完成安装
  • 因为hexo是基于node.js的，所以还需要安装node.js，同样很简单
  • 根据官网描述，hexo的安装是基于Git和Node.js的，既然有了以上的步骤，那就很简单了

至此，基础准备完成！
